# vita

note taking app

### TOOLS

* `GNU Make`
* `go 1.4+`
* [gb](https://github.com/constabulary/gb/) as go build tool
* [go-bindata](https://github.com/jteeuwen/go-bindata) to embed UI resources into executable file
* `Node.js`

* Optional `bower` if you want to update fonts or icons
